# Как добавить плагины в магазин (Сервер Обновлений)

Магазин плагинов в Spraute Studio настроен на загрузку с вашего выделенного сервера: `http://85.239.59.203/plugins/`.

Чтобы сообщество увидело и смогло скачивать новые плагины (или их обновления), вам нужно загрузить файлы `.splugin` и файл списка `market.json` на сервер.

## Шаг 1. Файл плагина (.splugin)
Когда кто-то создал крутой плагин (или вы сами), он экспортируется из Студии напрямую в формате `.splugin`. 
Вам **не нужно** ничего переименовывать или сжимать вручную — формат `.splugin` уже является готовым архивом, и Студия умеет скачивать и распаковывать его прямо с сервера.

Пример файла: `MyEpicCombat.splugin`

## Шаг 2. Загрузка на сервер
Загрузите файл `.splugin` на ваш сервер в директорию, откуда раздаются плагины. В коде студии настроено скачивание по такому пути:
`http://85.239.59.203/plugins/{fileName}`

Поэтому на сервере файлы должны лежать в папке `plugins`.

## Шаг 3. Обновление файла `market.json`
В той же папке `plugins` на сервере должен лежать файл списка магазина: `market.json` (`http://85.239.59.203/plugins/market.json`).

Откройте его и добавьте новую запись о плагине. Это обычный массив JSON-объектов:

```json
[
  {
    "name": "MyEpicCombat",
    "author": "ZonarStudio",
    "desc": "Самый крутой плагин для создания боссфайтов и PvP арен.",
    "version": "1.0.0",
    "mcVersion": "1.20.1",
    "fileName": "MyEpicCombat.splugin"
  },
  {
    "name": "AdvancedDialogues",
    "author": "ReadixGG",
    "desc": "Сложные ветвящиеся диалоги с выбором ответов.",
    "version": "1.2.5",
    "mcVersion": "1.19.4",
    "fileName": "AdvancedDialogues_v1.splugin"
  }
]
```

### Важные поля в `market.json`:
- **`name`** (Обязательно) — Название плагина. Должно в точности совпадать с `name` внутри файла `plugin.json` самого плагина (чтобы Студия понимала, установлен ли он уже у пользователя, и могла проверять версию для обновления).
- **`author`** — Имя автора плагина.
- **`desc`** — Краткое описание плагина.
- **`version`** — Текущая версия (Например, `1.0.0`). Если у пользователя установлена версия ниже, кнопка "Скачать" превратится в "Обновить".
- **`mcVersion`** — Версия Minecraft, для которой предназначен плагин (например, `1.20.1`). Она будет красиво отображаться в магазине.
- **`fileName`** (Обязательно) — Точное имя файла (например, с расширением `.splugin`), который загружен на сервер рядом с `market.json`.

## Как другим людям добавлять свои плагины?
В самой студии, в окне магазина, добавлена кнопка со ссылкой: **"Как добавить сюда свой плагин?"**.

Она ведет на ваш пост в Telegram: [https://t.me/spraute_community/83357](https://t.me/spraute_community/83357). 

Пользователи будут переходить туда и скидывать вам в комментарии экспортированные плагины (`.splugin`), а вы (как администратор) можете проверять их, загружать на сервер и добавлять строчку в `market.json`. После этого плагин мгновенно появится в Студии у всех пользователей!
